        How to use it

        Space Note App is a versatile platform that offers a simulated environment for various types of note-related projects, providing notes, categories,archived,favourite, and user data in JSON format. It serves as a valuable resource for testing and prototyping a real database. You can leverage Space Note App to enhance your development, learning, and testing endeavors.
        Here are some key features and example endpoints you can utilize within Space Note App for your projects:

        Users

        Create user 

        							
	fetch('https://space-note-app.onrender.com/user',{
		method:"POST",
		body:JSON.stringify(
			{
				email:'text@mail.com',
				password:"password"
			}
		)
	})
	.then(res=>res.json())
	.then(json=>console.log(json))

        Gat all users 

        								
	fetch('https://space-note-app.onrender.com/user')
	.then(res=>res.json())
	.then(json=>console.log(json))
